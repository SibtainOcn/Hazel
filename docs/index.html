Universal Compatibility

Engineered with a robust, custom-tuned extraction backend, Hazel provides seamless media retrieval from over 3,000 supported websites and web services.

Advanced Audio Processing

Utilize integrated processing algorithms for lossless and high-quality on-device audio transcription to formats like MP3, FLAC, WAV, and Opus.

Intelligent Management

Built for heavy media workloads. Queue multiple streams, manage parallel background downloads, and utilize auto-resume capabilities without draining system resources.

Built on Trust. Completely Open Source.

Hazel operates entirely on-device, ensuring your media habits remain strictly private. No telemetry, no data collection, no ads, and no hidden subscriptions. The code is 100% transparent and available for anyone to verify.

Hazel is Free and Open Source Software (GPL-3.0).
